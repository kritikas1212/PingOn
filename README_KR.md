## 🚀 시작하기

<div align="center">

```bash
# 저장소 복제
git clone https://github.com/sioaeko/statuswatch.git

# 디렉토리 이동
cd statuswatch

# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build
```
